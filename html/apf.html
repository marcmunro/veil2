Change History
Version git tag Release Date Notes

Major update to:

  • make Veil2 a Postgres extension;

Update to:

  • improve documentation;
  • improve performance;
  • improve testing, and fix bugs;
  • improve the pgxn metadata and rendering of docs.

It is believed that this version will be stable. There are no planned enhancements or updates at the time of release.

Consider this a production release candidate.


Veil2 will be upgraded to production status once it has proved to be effective and free of serious deficiencies.
