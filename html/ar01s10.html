10. Setting Up A Veil2 Virtual Private Database - Overview

Setting up your database to use Veil2 is a fairly straightforward process, and can be done in the following stages:

  1. install the Veil2 scripts;
  2. identify and define your scope types;
  3. choose and implement your authentication and session management mechanisms;
  4. create an initial set of privileges;
  5. integrate and/or create roles;
  6. create foreign key links for veil2.accessors;
  7. link your scopes and security contexts;
  8. create scope promotion mechanisms;
  9. secure your tables;
  10. secure your views;
  11. assign initial roles to users;
  12. test

Installation is covered in this previous section. The rest follows.
